# Smart Farming Platform Prototype

An AI-driven smart farming platform that uses real-time data to optimize resource usage, provide eco-friendly treatment recommendations, and instantly identify crop diseases.

## Features
- AI-based image-based crop disease detection (mock implementation)
- Recommendations for low-chemical, sustainable treatment
- Climate-conscious guidance for preventive agriculture
- User-friendly web interface

## Technologies
- Backend: Python Flask
- Frontend: HTML, CSS, JavaScript
- AI/ML: Mock CNN-based disease detection
- APIs: OpenWeatherMap for weather data

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python backend/app.py`
3. Open http://localhost:5000 in your browser

## Usage
- Upload crop images for disease detection
- View treatment recommendations
- Check climate guidance
